# Usage examples

## Generate a new module

```text
/amiga-protracker-mod-generator "Title: Machine Riot; genre: 1980s synth-pop dance with rave and drum-and-bass influences; 128 BPM; E minor; aggressive and urban; about 2 minutes; straight 4/4"
```

## Validate an existing module

```text
Use the amiga-protracker-mod-generator skill to inspect song.mod. Produce a binary validation report, sample and loop checks, period/effect checks, PAL/NTSC pitch analysis, and an approximate preview WAV without modifying the original file.
```

## Repair a module

```text
Use the amiga-protracker-mod-generator skill to repair broken.mod. Preserve the musical intent, correct structural errors and invalid loops, regenerate only unusable samples, and return both the repaired module and a detailed change report.
```
